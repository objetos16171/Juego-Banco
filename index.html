Aplicacion Tarjeta Bancaria

Es una aplicacion que le da un nuevo sentido al comprar alguna cosa por intenet, el usuario entrara a una plaza comercial, donde se estaran tres locales donde el usuario se podra desplazar y realizar compras, estos lugares son: una tienda de abarrotes, una tienda de ropa y una tienda de videojuegos. https://github.com/objetos16171/Juego-Banco/blob/master/centro.png

Las Tiendas

Dentro de las tiendas el usuario podra elegir varios podructos y pagarlos directo con su tarjeta de credito pero para esto cada tienda tiene un inventario y solo se podran comprar cierta cantidad de productos y las tarjetas tienen un limite de pago, si el usuario excede ese limite, se le cancela la compra al usuario y se reporta la tarjeta.

Utilizar la aplicacion

Para poder utilizar el usuario debe utilizar los cursores para poder entrar a cualquier tienda del centro comercial, ya despues de que esta dentro de la tienda debera utilizar el mouse para seleccionar lo que desea comprar y la compra se confirma cuando sale de la tienda.
